Item Inventory 관리

Item의 속성
-번호, 제목, 파워, 최대사용횟수, 현재 사용횟수
메모장을 이용하여 위와 같은 item 정보를 5개 저장하고 item.txt에 저장한다.

프로그램을 시작하면 다음과 같은 메뉴가 표시되도락 한다.
아이템 리스트(L), 아이템 선택(번호), 보유한 아이템 보기(s), 아이템 사용(u)

각 메뉴 구현
- 아이템 리스트: 파일에 등록된 아이템 리스트를 보여준다
- 아이템 선택(번호): 아이템 목록을 보면서 번호를 선택하면 보유한 아이템으로 등록된다.
- 보유한 아이템보기(s): 이용자가 현재 보유중인 아이템 리스트를 보여준다.
- 아이템 사용(u): 보유 중인 아이템을 사용하면 해당 아이템의 사용횟수가 증가하고 표시
- 최대사용횟수를 초과하면 해당 아이템은 더 이상 사용할 수 없다.

화면에는 항상 "현재 사용중인 아이템"과 보유중인 아이템 리스트가 보여야 한다.
현재까지 사용횟수도 표시되어야 한다.
